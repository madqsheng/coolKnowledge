# 1. 运动学

## 1.1 参考系

- 背景与动机
  - **基本事实**：*自然界的一切物质都处于永恒的运动中，绝对静止的物体不存在。*
  - 通俗说，运动从来不是绝对的。
  - 很多对于运动的描述，之所以看起来或者说起来是绝对的，是因为人眼看到的世界是经过大脑处理，自动选择了一个参考物来描述运动。其描述的运动本质是相对的。
  - 常见的运动参照物是地球
  - **透过现象看本质**：对被描述成绝对运动的现象，解析参照物，转换本质的相对运动。
- 定义：被参照用来描述某一种运动的物体。也就是上述的参照物

# 2. 力学

## 2.1 牛顿三定律

- **牛顿第一定律**：（**力学和运动学，注意需要考虑运动学中的参考系**）

   *一切物体总保持匀速直线运动状态或静止状态，除非作用在它的上面的力迫使改变这种状态*

- **牛顿第二定律**：（**力学和运动学，注意需要考虑运动学中的参考系**）

  *物体加速度的大小跟它受到的作用力成正比，跟它的质量成反比，加速度的方向跟作用力的方向相同。*

  - 数学表达式：
    $$
    F = ma
    $$

  - 单位：

    1. 力：牛顿 N
    2. 质量：千克kg
    3. 加速度：m/s

- **牛顿第三定律**

  *两个物体之间的作用力和反作用力总是大小相等，反向相反，作用在一条直线上*

# 3. 功与能

## 3.1 功

- 定义：**物体在力的作用下发生位移，力对物体所做的功，等于力的大小、位移的大小、力与位移夹角的余弦这三者的乘积。**

- 公式：
  $$
  W=Flcosα
  $$
  
- 单位：

  1. 功：焦耳J
  2. 力：牛顿N
  3. 位移：米m

- 正功与负功

  - 力和位移分别是力学和运动学里的概念，都是矢量，有方向
  - 因此力对物体做功可以是负功

## 3.2 功率

- 功率，做功效率，做功的快慢

- 公式：
  $$
  p=\frac{W}{t}=Fv
  $$

- 单位：

  1. 功率：瓦特w

  2. 功：焦耳J
  3. 时间：秒s
  4. 力：牛顿N
  5. 速度：m/s

- 解析公式：

  - 定义上，功率是功与时间的比值
  - 推导上，功率是力和速度的乘积

## 3.3 能

- 概述
  - 不得不说，能量其实是一个很抽象的概念
  - 让能量抽象，最大元凶是**势能（比如重力势能，电势能）**，以及**功和能量**的关系
- 理解：
  1. **能量是物质的一种属性，功是物质施加或者被施加力出现运动的的付出的代价**
  2. 这个代价就是能量，==外部的力对物体做功会让物体的能量变化==
  3. 因此能量和功是一个单位，都是焦耳
  4. 最重要的，能量从来不是绝对的，**能量是相对的概念**
  5. 正因为能量是相对的，谈论能量，往往是**有没有能量，能量变化了多少**，如果非要定量说**能量有多少，必须有一个参考量**
  6. **势能**，某种趋势（场）产生的能量，前缀就是来源，**重力势能，弹性势能（非场），电势能**，当然也有**磁力势能**等
  7. **能量不是实体，不是物质。能量是物质的属性。**
- 能量相对性举例：
  1. **动能是最好理解的，有质量和速度，直觉上也知道危险，因为有能量。但速度本身是相对的，因此动能取决于参考系**
  2. **重力势能跟质量和高度有关，高度本身是相对的。弹性势能跟形变大小有关，形变同样是相对的。**
  3. **电势能同样是相对的，其定义本身就是相对性的描述：将某点电荷从某点移动到参考点时电场力做功称为该电荷在该点的电势能**

### 3.3.1 能量分类

- 按照领域分：
  1. **机械能**：机械运动产生的能量，包括：**动能，重力势能，弹性势能**
  2. **核能**
  3. **化学能**
  4. **生物能**
  5. **内能（热能）**
  6. **电能（电势能）**
  7. **辐射能**
  8. **磁能（磁力势能）**
  9. **光能**
  10. **辐射能（电磁波能）**
- 按照性质
  1. **动能**
  2. **势能（场能）**

### 3.3.2 能量变化与做功

- 统一说法：

  **物体受到力的作用，有了运动（位移），物体的能量会变化。**

- 两种能量，两种情况：

  1. **物体动能的变化，等于作用在物体合力的做功**
  2. **物体势能的变化，等于作用在物体的对应场力的做功。**

# 4. 电荷

- 概述
  - electricity
  - **电荷并不是电子，不是中子，也不是离子**
  - **电荷从来不是物质，而是物质的属性，是一种物理量**
  - 电荷特别的地方在于，除了有大小，还要正负

## 4.1 导电性与自由电子

- 本质上，导电性是因为物质里存在**自由电子**

1. 自由电子

   **能脱离原子的束缚，而自由运动的电子**

2. 离子

   **带电的原子**，离子不会自由运动，而是在内部排列起来，在自己的平衡位置附近振动而不移动

- **问题1：金属都导电吗？**

  是。注意金属物质本身是金属原子直接通过原子构成，某这意义上说，金属物质可以用金属原子来分析。金属原子的外层电子很容易挣脱束缚，形成自由电子。

- **问题2：金属性和导电性是一个概念吗？**

  不是。金属性是微观世界里原子的性质，导电性是宏观世界物质的性质。金属性是原子对键合电子的作用力。导电性是物质里自由电子的数量。

- **问题3：还要导电的非金属物质吗？**

  有。比如电解质溶液，溶液里的阴阳离子可以运载电荷移动。石墨，共价键里有游离的电子。还要**半导体**。

## 4.2 库伦定律—电荷之间的力

- 库仑定律：*真空中两个静止点电荷之间的相互作用力，与它们的电荷量乘积成正比，与它们的距离的二次方成反比，作用力的方向在它们的连线上。*

- 表达式：
  $$
  F = k\frac{q_1q_2}{r^2}
  $$

  - 单位：
    - 力：牛顿N
    - 电荷量：库仑C
    - 距离：米m
    - 比例系数k：常量 k=9.0x10^9^N·m^2^/C^2^

# 5. 半导体

- 概述
  - 半导体是一种材料，是**具有某种性质的物质**的总称
- 性质
  1. 导电性介于导体和绝缘体
  2. 单向导电
- **半导体不等于硅**，以硅为主要成分的半导体，其实是混合物，因为需要“掺杂”来提供自由电子和正电荷
- 半导体分类：
  1. 化合物半导体。两种或两种以上元素化合而成。**纯净物**。它的种类很多，重要的有砷化镓、磷化锢、锑化锢、碳化硅、硫化镉及镓砷硅等。
  2. 元素半导体。比如：锗、硅、硒、硼、碲、锑，其中用**硅制造的半导体器件，耐高温和抗辐射性能较好，特别适宜制作大功率器件**。注意，化学性质的原因，**自然界几乎没有硅单质**，也就是硅原子通过共价键形成物质。**单单是硅原子构成的物质也没办法单向导电**。**“掺杂”形成的PN结构二极管才具备单向导电性，即二极管等半导体器件是混合物**。
  3. 有机增导体材料

## 5.1 单向导电性

- **电子和空穴**
  - 电子很简单，无需多言
  - 空穴直觉上并不难，共价键里有个电子挣脱了，留下了轨道里留下了空地。抽象的点在于，半导体里认为**空穴也在动**

- 硅元素为主要成分的半导体材料

  1. 理想情况下，硅单质通过共价键作用，内部很稳定，电子很难移动，很难成为自由电子

     ![硅单质](..\实例图片\硅单质.jpg)

     

  2. “掺杂操作”：在硅里掺杂元素P（磷）和B（硼），磷是正五价，硼是正三价。分别形成**N型半导体**和**P型半导体**

     ![硅单质掺杂p和b](..\实例图片\硅单质掺杂p和b.jpg)

     - 注意，

       **掺杂P（磷）原子和硅原子形成化合价，会多出来很多电子，就有了更多自由电子。形成N型半导体材料**

       **掺杂B（磷）原子和硅原子形成化合价，会多出来很多空穴。形成P型半导体材料**

       

  3. P型和N型半导体放一起

     ![P型和N型半导体放一起](C:\Users\admin\Desktop\coolShare\实例图片\P型和N型半导体放一起.jpg)

     - 两者型号的半导体材料中的电子和空穴浓度（concentration gradient）不一样，会出现**扩散**（diffusion）

     - n型中电子往p型扩散，p型中的空穴往n型扩散

     - 扩散的结果是两种型号的半导体里的原子失去电平衡，开始变成**离子**，**带电**

     - 带电的结果是形成**电场**

     - 如果不明白**场**的概念，记住，场是一种物质，它是产生非接触力的原因。比如静电力，重力，磁力等

     - 形成电场的结果是，会有一个静电力，对电子和空穴的**静电力方向和扩散方向相反**。

     - 结果是扩散会到达一个平衡。形成一个**耗尽区（Depletion Region）**。

       

  4. 外部加入两个方向的电场

     ![P型和N型半导体加入电场](..\实例图片\P型和N型半导体加入电场.jpg)

     - **电池可以提供电压，即电池正负极分别带正负电荷，电荷自然有电场，也可以说，电池提供电场**
     - 外部电场方向两种情况：
       1. 外部电场和内部电场相反，结果是耗尽区变窄。操作是电池正极连接P型
       2. 外部电场和内部电场相同，结果是耗尽区变宽。操作是电池正极连接N型
     - 耗尽区变化的结果就是导电性变化。分析如下：
       - 耗尽区本质是一个电场区，对电子和空穴的**静电力方向和扩散方向相反**
       - 这样一来，耗尽区的电场是要阻止电子和空穴的扩散（源于浓度梯度）
       - 电子和空穴不移动了，自然没有电流，导电性就很差了
     - 电池提供的外部电场就是要推动半导体内部的电子和空穴。
     - 外部电场的方向衍生两种思路：
       1. **正向偏置**。电池正极连接P型，**外部电场推动内部电子和空穴对抗内部电场**。耗尽区变窄，**导电性提升**。
       2. **反向偏置**。电池正极连接N型，**外部电场和内部电场同反向**。耗尽区变宽，**导电性变差**。
     - **电池连接方向不同，导电型截然不同。这种现象就是半导体最重要的单向导电性。**

## 6.1 电场

- 概述

  - 电荷之间的库仑力源于电场，因为电荷周围存在电场。

- **静电感应**

  - *当带电体靠近导体时，由于电荷之间的吸引和排斥，导体中的自由电荷便会趋向或者原理带电体，使导体靠近带电体的一端带异种电荷，远离带电体的一端带同种电荷。*
  - 摩擦起电，会得到一个带电体，带电体靠近轻微物体，比如纸屑，会吸引纸屑，原因是感应起电。

- 表达式
  $$
  E = \frac{F}{q}
  $$

  - 单位：
    1. 电场强度：牛每库 N/C
    2. 电荷量：库仑C
    3. 力：牛顿 N
  - **方向：电场方向等于正电荷在电场中受到静电力的方向**

### 4.1.1 电势能

- 强调理解
  1. **能量是物质属性**
  2. **能量是相对的**
  
- 定义：
  
  *将某点电荷从某点移动到参考点时电场力做功称为该电荷在该点的电势能*

### 6.1.2 电势

- 定义：

  *电势是电荷在电场中某一点的电势能和电荷量之比*

- 表达式：
  $$
  φ = \frac{E_p}{q}
  $$

  - 单位：
    1. 电势：伏特 V
    2. 电势能：焦耳 J
    3. 电荷量：库仑 C

- 性质：

  **沿着电场方向，电势逐渐降低**

- 另一个角度：

  1. **电势能是电荷（带电体）的属性。**
  2. **电势是电场的属性。从电势能到电势，已经剥离了电荷。**
  3. **电势和电势能都是相对量**

### 6.1.3 电压=电势差

- 理解：
  1. **电场产生的力对电荷做功，如果是正功，电荷的电势能减少。如果是正功，电荷的电势能增大。**
  2. **电势差是一个绝对量，只和电场做功的多少和正负有关**

- 表达式：
  $$
  U_{AB} = \frac{E_{pA}-E_{pB}}{q}=\frac{W_{AB}}{q}
  $$

- **电场计算电势差**：
  $$
  U_{AB} = \int {E(s)ds}
  $$

### 6.1.4 电池提供电压（电势差）

- 整体总结：
  1. 电池的正负极会带电，正极带正电，负极带负电
  2. 直接用导线连接电池会短路，烧坏电池
  3. **用导线只连接电池的一端，其实导线也带电了。实验用导线去吸附轻微物体，看是否有静电感应。**
  4. 总结：==用导线连接电池正负极，导线两端带正负电，自然会有电场，自然有电势差==

-  **原电池原理**：

  - **氧化反应**和**还原反应**
    - 氧化反应：物质所含元素**化合价升高**的反应。
    - 还原反应：物质所含元素**化合价降低**的反应。
    - 氧化还原反应：物质中有元素化合价升高和降低的反应。

  - **原电池**是通过氧化还原反应而产生电流的装置

    ![原电池原理](..\实例图片\原电池原理.png)

- 关键概念：

  - **电解质**：在**水溶液**或**熔融状态**下能够导电的化合物，如**盐酸HCl**、**硫酸H~2~SO~4~**、NaOH、NaCl等。**导电原因是有自由电子或者离子。**

    ==常说水导电，其实所谓的水是混合物，里面电离了很多离子。纯净水是不导电的==

  - **电离**：电解质溶于水或受热融化时，形成**自由移动的离子**的过程

- 分析

  - 稀硫酸H~2~SO~4~，其实是混合物，里面有水。
  - H~2~SO~4~（包括HCl）本身是共价化合物，原子之间是共价键。
  - H~2~SO~4~是电解质，溶于水后，会电离出H^+^离子，SO~4~^2-^离子
  - 其实**锌和铜都可以和H^+^离子反应来失去电子**。但是**锌金属性更强**，更容易失去电子。
  - 正负极会发生反应，溶液里的离子会出现定向移动，就会有电流：
    1. 负极：锌棒**失去电子**，通过外部导线浓度扩散移动到正极。
    2. 正极：铜棒有**更多电子**，吸引溶液的**H^+^离子移动到正**极去发生反应。

- **原电池形成条件**

  1. 电极材料由两种**不同活泼性的金属**或其他**导电**材料(非金属或某些氧化物等)组成。
  2. 电解质
  3. 电线连接在两个电极之间形成一个闭合电路
  4. 发生的反应是自发的氧化还原反应

- **原电池常见电极**

  1. 活泼性不同的金属：如锌铜原电池，锌作负极，铜作正极
  2. 金属和非金属(非金属必须能导电)：如锌锰干电池，锌作负极，石墨作正极
  3. 金属与化合物如：铅蓄电池，铅板作负极，二氧化铅作正极
  4. 惰性电极如：氢氧燃料电池，电极均为铂
