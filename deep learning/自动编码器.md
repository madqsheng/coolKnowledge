# 概述

- 模型结构

  自动编码器有两部分：**编码器和解码器**。编码器减少输入数据的维度，使得原始信息处于压缩形式。解码器从压缩数据中恢复原始信息。

![自编码器](..\示例图片\自编码器.jpg)

- 应用
  - 数据去噪
  - 为进行可视化而降维
